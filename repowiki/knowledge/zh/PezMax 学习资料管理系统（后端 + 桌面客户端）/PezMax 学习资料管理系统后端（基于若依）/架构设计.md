采用 Maven 多模块聚合工程，根 pom.xml 通过 dependencyManagement 统一版本，modules 声明 ruoyi-admin / ruoyi-framework / ruoyi-system / ruoyi-quartz / ruoyi-generator / ruoyi-common / ptmj-datum 七个子模块，依赖方向自顶向下：
- ruoyi-admin：应用入口 RuoYiApplication.java，@ComponentScan 同时扫描 com.ruoyi 与 com.ptmj，排除 DataSourceAutoConfiguration 由 framework 接管；内含 web.controller.common（验证码/文件上传）、web.controller.datum（ptmj 业务控制器）、monitor/system/tool 等若依自带控制器。
- ruoyi-framework：安全与基础设施层，SecurityConfig + JwtAuthenticationTokenFilter 实现 JWT 鉴权，DruidConfig/RedisConfig/MyBatisConfig/ResourcesConfig 等集中装配，aspectj 下 DataScope/Log/RateLimiter/DataSource 切面，web.exception.GlobalExceptionHandler 统一异常处理。
- ruoyi-system：系统内置 RBAC 领域（SysUser/SysRole/SysMenu/SysDept 等），提供 ISysUserService 等接口及 MyBatis XML。
- ruoyi-quartz：Quartz 定时任务管理（SysJob/SysJobLog）。
- ruoyi-generator：Velocity 模板代码生成器。
- ruoyi-common：通用注解（@DataScope/@Log/@RepeatSubmit/@Sensitive）、R/AjaxResult/BaseEntity 基础域、MinioConfig/MinioStorageService 对象存储封装、XSS 过滤器、工具集。
- ptmj-datum：业务核心，domain/mapper/service(impl) 完整分层，mapper XML 位于 resources/mapper/datum/*Mapper.xml，并通过 springdoc-openapi 暴露 REST API；额外提供 PtmjFileRankCacheService/PtmjFileTreeCacheService 两个 Redis 缓存服务。

运行期通过 compose.yaml 编排 MySQL + Redis + MinIO，application.yml 中 minio.url/accessKey/bucketName 指向容器内 minio:9000，ptmj.file.type-map 定义资料分类映射，springdoc 默认开启 /swagger-ui.html。
